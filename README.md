# 作りたいシステムの仕様
LEDが光ったら対応するキーボードをなるべく早く押す。
早く押すほど得点が高い。
高い得点を目指す。
パターンをスイッチで切り替える
LEDをブレッドボードにつないで出力の光を大きくする

LED: 次の2つの目的で使う.
光る用
キーボード押したときに光る用

ソフト
使用言語: C言語
キーボードの入力を0, 1に変換してパターンのメモリに送り込む.
回路から与えられたメモリから得点を取得して記録する.
