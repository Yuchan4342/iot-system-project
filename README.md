# 作りたいシステムの仕様
LEDが光ったら対応するキーボードをなるべく早く押す。
早く押すほど得点が高い。
高い得点を目指す。
パターンをスイッチで切り替える
LEDをブレッドボードにつないで出力の光を大きくする

LED: 光る用とキーボード押したときに光る用を用意

LEDを光らせるAPI: User manual.
